Keyless Transposition



Read More

A transposition cipher does not substitute one symbol for another instead it changes the location of the symbols. Keyless Transportation cipher is simple one which is keyless. In first method the text is written into a table column by column and then row by row. In the second method the text is written into the table row by row and then transmitted column by column. Example of a Keyless transposition cipher using first method is rail fence cipher.
